Discord Blackjack Bot

Description

A Discord bot written with GO, that allows users to play blackjack one-on-one against the bot. Players area given a starting sum of chips, and are able to wager chips on each game against the bot. Players can never drop below one chip, so they are always able to continue playing. The bot tracks stats on the players so they can view their wins, losses, ties, and total chips.


This project was developed as a way to learn the programming language GO, as well as learning to develop a bot for Discord. Uses Discord application commands and button components to play.

Development Process

I began the development process by learning the programming language GO. I was brand new to GO and did not have any prior knowledge of the language. I chose to first follow Google's tutorial called "A Tour of GO", which gives a quick run-down of all of the major features of GO. This tutorial does include some exercises, but I found that I required more hands-on experience with the language to really learn it.


This lead me to the website Exercism, where I followed their GO learning track. This served as an excellent way to reinforce the concepts I learned in A Tour of GO with a variety of hands-on exercises using each core concept.


Once I was confident in my ability to code with GO, I began researching programming a Discord bot. I found a great library for GO that acts as a wrapper for the Discord API, called discordgo. GO is not one of the popular lanugages used to develop Discord Bots, so there were not a lot of resources available to learn how to use the library. Fortunately, the developer of the library included detailed examples in the repository that demonstrate how to use the library.


The development process of the bot was iterative. The first iteration was simply the core gameplay, done through text messages in Discord. Each subsequent iteration improved the user experience, the gameplay, and the structure of the code. I began exploring more of the Discord API and finding cool features that I wanted to implement in my game. I reworked the bot to use Discord Application Commands, as well as buttons, to improve the experience for the user.

Reflections

This project was very enjoyable to work on. I loved seeing the improvements between each iteration of the project, and my skills with both GO and the discordgo library improving as the project was developed. I plan to continue working on this project in the future and add more features, such as additional casino and card games. Also, now that I am able to create applications in Discord, next time I think of a good idea for one I will be able to create it myself!
