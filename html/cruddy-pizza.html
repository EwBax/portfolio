Cruddy Pizza

Description

An Android application that is a mock ordering system for a Pizza restaurant. Orders created are saved to a SQLite database, and all saved orders can be viewed and updated/deleted from within the appliation. The application can also switch between English and French the fly, and remembers the language selection.


This application was a school assignment to practice using Android Shared Preferences, and performing CRUD (Create Read Update Delete) operations with a database. The program was written with Java and developed using Android Studio.

Development Process

This assignment was part of my Android development course, and was the final assignment of the course. Before this point in the course, I had already developed a calculator app and a quiz app, but had not made an application with navigation back and forth between pages (activities). This didn't end up being the tricky part of the assignment, though; that would be the translation between English and French at the push of a button in the app.


I built the language switching functionality using the built in localization functionality in Android. Android's Localization will choose which resource dictionary to use for text in the app based on the locale of the device using the app. I programmed the button to switch the app to use the French locale resources and reload the current page, but for some reason it would still display English. I spent some time researching and testing possible solutions to my problem, but in the end the best approach I found was to have a function that manually sets all the text on the page with the new locale's resources.


To handle the CRUD operations in the app, I wrote a java database adapter and an order model to store the data for individual orders. To display the order history I used a custom XML control for each row displayed in a RecyclerView.
